# 🧠 HIRA AI
### Human Intelligence Recruitment Assistant

> **AI-Powered Intelligent Candidate Discovery & Ranking Platform**

HIRA AI is an AI-driven recruitment platform that helps recruiters identify the best candidates by understanding resumes and job descriptions semantically rather than relying on traditional keyword matching.

Instead of filtering resumes solely based on keywords, HIRA AI leverages **Google Gemini AI**, intelligent skill matching, and explainable ranking to recommend the most suitable candidates for any role.

---

# 🚀 Problem Statement

Recruiters review hundreds of resumes for every job opening.

Traditional Applicant Tracking Systems (ATS):

- ❌ Depend on keyword matching
- ❌ Miss qualified candidates
- ❌ Cannot explain rankings
- ❌ Ignore candidate potential

HIRA AI solves these challenges by providing an explainable AI-powered recruitment assistant.

# 🧠 AI Workflow

### Step 1
Upload multiple candidate resumes.

### Step 2
Paste the Job Description.

### Step 3
Gemini AI extracts:

- Skills
- Experience
- Education
- Projects
- Certifications

### Step 4
Job requirements are analyzed.

### Step 5
Candidates are matched using intelligent skill comparison.

### Step 6
Ranking engine calculates suitability scores.

### Step 7
Dashboard displays:

- Candidate Ranking
- Match Percentage
- Missing Skills
- AI Recommendations
- Hidden Gem Candidates

### Step 8
Export ranked candidates as Excel.

---

# 📊 Scoring Methodology

The final score is calculated using multiple evaluation signals:

- Technical Skills
- Programming Languages
- Framework Knowledge
- AI / ML Skills
- Cloud Technologies
- Experience
- Education
- Missing Skills Penalty

This provides a more reliable candidate ranking than simple keyword filtering.

---

# 💎 Hidden Gem Detection

HIRA AI identifies candidates who may not rank first but demonstrate strong potential.

A Hidden Gem typically has:

- Moderate overall score
- Strong core skills
- Few missing requirements
- High interview potential

This helps recruiters discover overlooked talent.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Suvasini911/HIRA-AI.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Backend

```bash
uvicorn backend.main:app --reload
```

Run Frontend

```bash
streamlit run frontend/app.py
```
